file index akan menampilkan halaman utama website dalam file index terdapat pengertian dari beasiswa yang tersedia

file daftar.php akan menampilkan halaman form pendaftaran beasiswa pada halaman ini pengguna di haruskan untuk mengisi
data yang dibutuhkan lalu apabila data sudah terisi tekan tombol daftar untuk menyimpan data pada database untuk
ditampilkan pada halaman hasil

file hasil.php akan menampilkan data yang sudah di input pada halaman daftar pada file ini akan mengambil data yang ada
di database

file koneksi untuk menghubungkan program php ke database db_beasiswa
file get_ipk.php untuk menampilkan ipk apabila email sudah di input
file functions.php akan menambahkan file kedalam database
file grafik.php akan menampilkan grafik dari data yang ada
folder chartjs adalah library untuk membuat grafik
